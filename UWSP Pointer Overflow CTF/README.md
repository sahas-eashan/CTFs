# UWSP Pointer Overflow CTF Writeups

Single-source writeup log for every challenge solved during the UWSP Pointer Overflow CTF. Folders still store tooling, captures, and scripts, but the narrative stays here. Keep the sections in sync with your directory tree (`Category/Challenge-Name`).

## Workflow
- Create the challenge folder `Category/Challenge-Name` before or while investigating.
- Copy the template block below into the correct category and fill the fields as you work.
- Update the flag line only after validation; leave `flag{pending}` until ready.

```markdown
#### Challenge Name (Points)

- Summary: TODO short description of the solve path

	2. TODO supporting step
- Flag: `flag{pending}`

Notes:
- TODO observation
- TODO toolchain reminder
```
